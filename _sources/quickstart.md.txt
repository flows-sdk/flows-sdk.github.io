# Quickstart

Generate and deploy a sample Flow

1. Create a virtual environment and source it

    for example:
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```

2. Install the flows_sdk package

    ```bash
    pip install https://build-storage.int.hyperscience.com/flows_sdk/v0.1.2/flows_sdk-0.1.2-py3-none-any.whl
    ```

3. Build a sample flow using the `package_flow` executable that comes with the library:

    ```bash
    package_flow samples/hello_flow/flow.py --file-out hello.json
    ```

4. upload the produced `hello.json` to a running Hyperscience instance from `https://{hyperscience-url}/flows` -> Import Flow
